# 剧本杀H5网页应用开发计划

## 技术栈选择
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **实时通信**: Socket.io-client
- **UI组件库**: Ant Design Mobile
- **AI交互**: OpenAI API 或 本地大模型集成
- **文件处理**: Papa Parse (CSV), SheetJS (Excel), PDF.js (PDF)
- **样式**: Tailwind CSS

## 项目结构设计
```
JuBenKill/
├── public/              # 静态资源
├── src/
│   ├── assets/          # 项目资源
│   ├── components/      # 通用组件
│   ├── pages/           # 页面组件
│   ├── hooks/           # 自定义 hooks
│   ├── store/           # 状态管理
│   ├── services/        # API服务 (Socket.io, AI)
│   ├── utils/           # 工具函数
│   ├── types/           # TypeScript类型定义
│   └── App.tsx          # 入口组件
├── vite.config.ts       # Vite配置
└── package.json         # 项目依赖
```

## 核心功能模块
1. **用户系统**
   - 游客登录/账号注册
   - 用户信息管理
   - 房间创建/加入

2. **剧本管理**
   - 支持多种格式导入 (TXT, CSV, Excel, PDF)
   - 剧本解析与结构化
   - 剧本库管理

3. **AI主持人**
   - 剧情自动推进
   - 角色对话生成
   - 场景描述生成
   - 智能响应玩家提问

4. **游戏流程**
   - 角色分配
   - 剧本阅读
   - 搜证系统
   - 讨论环节
   - 投票系统
   - 结局判定

5. **多人联动**
   - 实时房间聊天
   - 角色状态同步
   - 动作同步 (搜证、投票等)
   - 房间管理

6. **交互场景**
   - 搜证地图与线索展示
   - 投票界面
   - 角色关系图谱
   - 时间
